# InternHub 🚀

A platform where students post their skills, and companies discover & hire them directly.

## ✨ Features

- **For Students**: Create profile, showcase skills, get discovered by companies
- **For Companies**: Search talent, direct messaging, hire without applications
- **Reverse Hiring Model**: Stop applying, let companies find you

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel
